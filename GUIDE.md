# Hermes Kanban — ツール概要 & 使い方ガイド

## これは何？

**Hermes Kanban** は、タスクをカード形式で管理する日本語対応のカンバンボードです。

- ブラウザだけで使える（インストール不要）
- データは自動でクラウド（Supabase）に保存される
- 複数のボードを切り替えて使える

**本番URL:** https://hermes-kanban.vercel.app

---

## 何ができるか

### カード管理
| 機能 | 説明 |
|------|------|
| カード作成 | タイトル・説明・優先度・担当者・期限・タグを設定 |
| カード編集 | クリックして内容を変更 |
| カード移動 | ドラッグ＆ドロップで列間を移動 |
| カード削除 | ホバー時に表示される🗑️ボタンで削除 |
| アーカイブ | 完了タスクを折り畳んで非表示にする |
| チェックリスト | カード内にサブタスクを追加・管理 |
| コメント | カードにコメントを追加 |

### ボード管理
| 機能 | 説明 |
|------|------|
| 複数ボード | プロジェクトごとにボードを作成・切替 |
| ボード作成 | 名前と絵文字を設定 |
| ボード編集/削除 | サイドバーからいつでも変更可能 |

### フィルター & 検索
| 機能 | 説明 |
|------|------|
| テキスト検索 | タイトル・説明・担当者名で絞り込み（ハイライト表示） |
| 優先度フィルター | 高・中・低で絞り込み |
| 担当者フィルター | 担当者名で絞り込み |
| 列ソート | 期限順・優先度順・作成日順に並び替え |

### その他
| 機能 | 説明 |
|------|------|
| ダークモード | ライト/ダーク切替 |
| 活動ログ | カードの作成・移動・削除の履歴を表示 |
| 期限通知 | 期限超過タスクをブラウザ通知で知らせる |
| 統計バー | 完了率をプログレスバーで表示 |

## 基本的な使い方

### 1. カードを追加する
- ヘッダーの **「追加」ボタン** または キーボードの **`N`** キー
- タイトル・優先度・担当者・期限を入力して保存

### 2. カードを動かす
- カードをつかんでドラッグ → 別の列にドロップ

### 3. ボードを切り替える
- 左サイドバーのボード名をクリック
- 新しいボードは **＋ボタン** で作成

### 4. 検索・絞り込み
- ヘッダーの検索バー（または **`/`** キー）でキーワード検索
- 優先度・担当者ボタンで絞り込み
- **`Esc`** キーでフィルターをクリア

---

## キーボードショートカット

| キー | 動作 |
|------|------|
| `N` | カード追加 |
| `/` | 検索バーにフォーカス |
| `Esc` | フィルタークリア |
| `⌘A` / `Ctrl+A` | 活動ログを開閉 |
| `⌘D` / `Ctrl+D` | ダークモード切替 |

---

## データの保存について

- **自動保存**: 操作するたびに自動でクラウドに保存されます
- **オフライン対応**: ネットが切れていてもブラウザ内に保存 → 復帰後に自動同期
- **データベース**: Supabase（PostgreSQL）に保存
